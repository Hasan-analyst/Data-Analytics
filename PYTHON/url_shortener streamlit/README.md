## Project Overview

This project is a **stylish URL Shortener web application** built using **Streamlit**.  
It allows users to convert long URLs into short, shareable links using the **TinyURL service**, with a clean UI and one-click copy functionality.

The application focuses on:

- Simplicity
- User-friendly design
- Practical use of Python libraries
- Interactive web deployment using Streamlit

---

## Features

- Paste and shorten long URLs instantly
- Generates short links using TinyURL
- One-click copy to clipboard
- Modern glassmorphism UI
- Responsive and clean design
- Fast and lightweight web app

## How the Application Works

1. User pastes a long URL into the input field
2. On clicking **Shorten URL**, the app sends the URL to TinyURL via `pyshorteners`
3. A shortened URL is generated instantly
4. The user can copy the shortened link with one click
5. Custom CSS provides a modern, visually appealing interface
