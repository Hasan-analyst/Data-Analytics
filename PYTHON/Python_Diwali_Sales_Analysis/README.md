## Project Overview

This project analyzes **Diwali sales data** using Python to uncover customer purchasing patterns, high-performing product categories, and revenue-driving customer segments.

The objective of the project is to perform **Exploratory Data Analysis (EDA)** on festive sales data and generate **business-oriented insights** that can help improve marketing strategies, customer targeting, and sales performance.

---

## Dataset Description

The dataset contains transactional sales data collected during the Diwali season.  
Each record represents a customer purchase and includes:

- Customer demographics (Gender, Age Group, Marital Status)
- Occupation and city category
- Product categories and product IDs
- Purchase amount
- Customer identifiers

The dataset is structured and well-suited for sales and customer behavior analysis.

---

## Business Problem

During festive seasons like Diwali, businesses need to:

- Understand customer buying behavior
- Identify high-revenue customer segments
- Analyze product category performance
- Optimize marketing and promotional strategies

This project addresses these challenges using data analysis techniques in Python.

## Analysis Performed

### Data Cleaning & Preparation

- Removed irrelevant or empty columns
- Handled missing values
- Converted data types for analysis
- Cleaned and standardized column names

---

### Exploratory Data Analysis (EDA)

#### Customer Demographics Analysis

- Purchase behavior by gender
- Spending patterns across age groups
- Analysis of marital status vs spending
- Occupation-wise purchase trends

**Insight:**  
Certain age groups and occupations contribute significantly more to total sales.

---

#### Regional Analysis

- Sales distribution across city categories
- Identification of high-revenue regions

**Insight:**  
Urban and metro regions show higher purchase amounts during Diwali.

---

#### Product Analysis

- Sales performance by product category
- Identification of top-selling product categories
- Revenue contribution by category

**Insight:**  
A small number of product categories generate a large share of total revenue.
